Acceptance Criteria
Không sửa code CREATOR-01 → CREATOR-14.
PostgreSQL + Drizzle ORM.
Khoảng 20 DB tables + enums được tạo và push thành công.
Repository, Service, Validator, Importer, Exporter và Runtime Bridge hoàn chỉnh.
Khoảng 60 REST endpoints tại /api/bosses/*.
16 React pages được tạo và route đầy đủ.
Sidebar có mục Boss Editor.
Tích hợp với NPC Editor, Skill Editor, Combat Editor, Dungeon Editor, World Editor, Runtime Engine và Asset Pipeline.
TypeScript strict, không phát sinh lỗi mới.
API Server và Creator Editor build thành công.
Cuối sprint xuất CREATOR-15 Sprint Report theo đúng format các sprint trước (Database, Backend, REST API, Frontend, Integration, Build Status, Acceptance Checklist)
