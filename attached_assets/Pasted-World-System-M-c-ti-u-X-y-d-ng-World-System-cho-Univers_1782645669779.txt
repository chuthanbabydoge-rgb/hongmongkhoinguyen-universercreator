Acceptance Criteria
Không sửa code CREATOR-01 → CREATOR-15.
PostgreSQL + Drizzle ORM.
Khoảng 20 DB tables + enums được tạo và push thành công.
Repository, Service, Runtime Engine và Managers hoàn chỉnh.
Khoảng 60 REST endpoints tại /api/world-system/*.
16 React pages được tạo và route đầy đủ.
Sidebar có mục World System.
Tích hợp với World Editor, Runtime Engine, NPC, Boss, Pet, Combat, Dungeon và Asset Pipeline.
TypeScript strict, không phát sinh lỗi mới.
API Server và Creator Editor build thành công.
Cuối sprint xuất CREATOR-16 Sprint Report theo đúng format các sprint trước.

Lưu ý: Nếu mục tiêu cuối cùng là một engine tương tự Roblox Studio/Unity, thì việc chuyển từ các "Editor" sang các "System" (World System, NPC System, Quest System, Combat System...) là hướng phát triển hợp lý. Các Editor tạo dữ liệu, còn các System chịu trách nhiệm vận hành dữ liệu đó trong runtime
